# 📝 Sistema de Gerenciamento de Tarefas em C

Um sistema simples mas completo de To-Do List desenvolvido em C para praticar arrays, structs, manipulação de arquivos e ponteiros.

## ✨ Funcionalidades

- ✅ Adicionar/Remover/Editar tarefas
- 📊 Estatísticas de progresso
- 💾 Persistência em arquivo
- 🎯 IDs automáticos
- 📱 Interface intuitiva no terminal

🎯 Resumo Prático
Biblioteca	Para que serve no projeto
stdio.h	Entrada/saída: mostrar menus, ler comandos, salvar/ler arquivos
string.h	Manipular textos: limpar o \n do título, comparar palavras
stdlib.h	Utilidades: NULL, system(), malloc(), funções de conversã




## 🛠️ Compilação e Execução
```bash 
gcc -o tarefas ListadeTarefas.c
./tarefas
